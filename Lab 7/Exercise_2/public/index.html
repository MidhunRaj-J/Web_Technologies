📚 Online Book Finder

Search, filter, and explore books using MongoDB queries

🔍 Module 1 — Search Books by Title

db.books.find({ title: { $regex: "...", $options: "i" } })

📄 Module 2 — Filter Books by Category

db.books.find({ category: "..." })

📈 Module 3 — Sort Books

db.books.find().sort({ price: 1 })

db.books.find().sort({ rating: -1 })

⭐ Module 4 — Top Rated Books

db.books.find({ rating: { $gte: 4 } }).limit(5)

📄 Module 5 — All Books with Pagination

db.books.find().skip(n).limit(5)
